## Исправленные баги в процессе тестирования

1. **lastInsertId() возвращал undefined** — исправлена функция `lastInsertId()` для корректной работы с sql.js
2. **Фронтенд не раздавался** — исправлен путь static middleware (`frontend` вместо `frontend/dist`)
3. **UNIQUE constraint при seed** — добавлена обработка ошибок вставки участников проектов

## Не тестировалось через API (требует ручной проверки в UI)

- Показ/скрытие пароля на форме входа
- Drag-and-drop на канбан-доске
- Адаптивность (mobile/desktop)
- Время загрузки страниц ≤ 2 сек
- XSS/CSRF защита
- Rate limiting на эндпоинтах авторизации
- Работа в разных браузерах (Chrome, Firefox, Safari, Edge)
